# PIC-Core

## Plataforma de Inteligencia de Compras

PIC-Core es una plataforma desarrollada para centralizar, normalizar y comparar listas de precios de múltiples proveedores.

## Objetivos

- Importar listas de precios
- Normalizar productos
- Construir un Catálogo Maestro
- Comparar precios
- Detectar cambios
- Generar reportes
- Automatizar procesos con IA

## Proveedores iniciales

- San Pablo
- Martínez
- Torres
